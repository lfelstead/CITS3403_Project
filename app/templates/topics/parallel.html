UnPARALLELed levels of learning

Resistors in parallel are connected next to each other in the same split circuit. That looks like this:

The formulas this time get a bit harder AND there is two of them. For a set of two resistors in parallel the formula is:

RT = R1 * R2⁄R1 + R2

And the formula for three or more resistors in parallel is:

1⁄RT = 1⁄R1 + 1⁄R2 + 1⁄R3 + ...

So, to calculate the resistance of this circuit:


The resistance is:

400Ω * 700Ω⁄400Ω + 700Ω = 254.54Ω


But to calculate this circuit:


The resistance is:

1⁄400Ω + 1⁄700Ω + 1⁄550Ω = 1⁄0.005746Ω = 174.01Ω
